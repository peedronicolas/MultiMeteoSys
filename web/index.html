-API REST de Meteorología

URLs de ejemplo:

-http://localhost:8080/apirest/meteorologia Para ver los ultimos registros meteorologicos desde la API Rest.

-http://localhost:8080/apirest/time?cod=json&est=murcia&time=4000 Para establecer el tiempo de envio en milisegundos (4, en este caso) de la estacion de Murcia desde la API Rest.

-http://localhost:8080/apirest/stop?cod=json&est=murcia Para pausar el envio de mensajes de la estacion de Murcia desde la API Rest.
-http://localhost:8080/apirest/start?cod=json&est=murcia Para reanudar el envio de mensajes de la estacion de Murcia desde la API Rest.

-http://localhost:8080/apirest/codeJSON?cod=json&est=murcia Para indicar a la estacion de Murcia que nos envie los mensajes usando JSON desde la API Rest.
-http://localhost:8080/apirest/codeXML?cod=json&est=murcia Para indicar a la estacion de Murcia que nos envie los mensajes usando XML desde la API Rest.

-http://localhost:8080/apirest/tempC?cod=json&est=murcia Para indicar a la estacion de Murcia que nos envie la variable temperatura en grados CELSIUS desde la API Rest.
-http://localhost:8080/apirest/tempK?cod=json&est=murcia Para indicar a la estacion de Murcia que nos envie la variable temperatura en grados KELVIN desde la API Rest.
-http://localhost:8080/apirest/tempF?cod=json&est=murcia Para indicar a la estacion de Murcia que nos envie la variable temperatura en grados FAHRENHEIT desde la API Rest.
